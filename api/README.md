# CRM API

Проста CRM система на ASP.NET Core.

## Можливості
- CRUD для клієнтів
- CRUD для замовлень
- Зв’язок: клієнт → замовлення
- Валідація (FluentValidation)
- Логування
- AutoMapper

## Технології
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- AutoMapper
- FluentValidation

## Запуск
1. dotnet restore
2. dotnet ef database update
3. dotnet run

Swagger:
https://localhost:xxxx/swagger

## Опис
Проєкт створений як pet-project для вивчення ASP.NET Core та побудови REST API.

# CRM API

A simple CRM system built with ASP.NET Core.

## Features
- CRUD for clients
- CRUD for orders
- Relationship: client → orders
- Validation (FluentValidation)
- Logging
- AutoMapper

## Technologies
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- AutoMapper
- FluentValidation

## Run
1. dotnet restore
2. dotnet ef database update
3. dotnet run

Swagger:
https://localhost:xxxx/swagger

## Description
This project was created as a pet project to learn ASP.NET Core and build REST APIs.
